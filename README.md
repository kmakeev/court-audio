# Аудиопротокол

Внешний модуль аудиопротоколирования судебных заседаний — надёжная запись
звука в зале/кабинете с привязкой к делу и выгрузкой в экспертную систему
[`ex_system`](../ex_system/) (обработка — модуль W2.11).

Функциональный ориентир — SRS «Фемида». В первой версии — **только аудио**;
видео и ГОСТ-ЭЦП — фаза 2.

## Состояние

v1 (этапы 00–08) реализована: захват, надёжность, хранилище/целостность, UI,
привязка к делу, выгрузка в `ex_system`, упаковка и дистрибутивы. Архитектура и
поэтапный план — в [`CLAUDE.md`](CLAUDE.md) и каталоге [`promts/`](promts/).
Журнал изменений — [`CHANGELOG.md`](CHANGELOG.md).

## Стек

Tauri 2.x (Rust-ядро + React-UI на дизайн-системе «Правовой кодекс»).
Кроссплатформенно: Astra Linux SE, РЕД ОС, Windows 10/11.

## Сборка

```bash
npm install
npm run tauri dev      # запуск в режиме разработки
npm run tauri build    # сборка дистрибутива под текущую ОС
```

Дистрибутивы по матрице ОС, оффлайн-установка в закрытом контуре, подпись,
обновления и сборка под Astra SE / РЕД ОС — [`docs/packaging.md`](docs/packaging.md).
Подготовка к Реестру отечественного ПО — [`docs/registry-checklist.md`](docs/registry-checklist.md).
