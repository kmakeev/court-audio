# Аудиопротокол

Внешний модуль аудиопротоколирования судебных заседаний — надёжная запись
звука в зале/кабинете с привязкой к делу и выгрузкой в экспертную систему
[`ex_system`](../ex_system/) (обработка — модуль W2.11).

Функциональный ориентир — SRS «Фемида». В первой версии — **только аудио**;
видео и ГОСТ-ЭЦП — фаза 2.

## Состояние

Проект в стадии планирования. Архитектура и поэтапный план — в
[`CLAUDE.md`](CLAUDE.md) и каталоге [`promts/`](promts/).

## Стек

Tauri 2.x (Rust-ядро + React-UI на дизайн-системе «Правовой кодекс»).
Кроссплатформенно: Astra Linux SE, РЕД ОС, Windows 10/11.

## Сборка

Скаффолд проекта выполняется по [`promts/00_infra.md`](promts/00_infra.md)
(пока не выполнен). После него — стандартные команды Tauri:

```bash
npm install
npm run tauri dev      # запуск в режиме разработки
npm run tauri build    # сборка дистрибутива под текущую ОС
```
