# Журнал изменений — «Аудиопротокол»

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
версионирование — [семантическое](https://semver.org/lang/ru/).

## [0.1.0] — 2026-07 (v1)

Первая версия модуля захвата: только аудио, оффлайн-захват с чанковой догрузкой
в `ex_system`. Итог этапов 00–08.

### Добавлено

- **00 · Инфраструктура.** Скаффолд Tauri 2 (Rust-ядро + React 19/PravoUI),
  схема `Settings` (зеркало `docs/configuration.md`), CI-матрица
  macOS/Ubuntu/Windows.
- **01 · Ядро захвата.** `cpal`: устройства, поток, кольцевой буфер, сегментный
  WAV-райтер (PCM), индикаторы уровня.
- **02 · Запись и надёжность.** Автомат сессии, непрерывный сброс + ротация,
  восстановление после сбоя/питания, watchdog, обрыв устройства, контроль диска.
- **03 · Хранилище и целостность.** SQLite-манифест, шифрование at-rest
  (AES-256-GCM), SHA-256 + хеш-цепочка сегментов, журнал событий, ретеншн.
- **04 · UI записи.** Экраны Запись · Сессии · Настройки · Диагностика на
  PravoUI; метры уровня, старт/стоп/пауза, индикатор сохранения.
- **05 · Привязка к делу (оффлайн).** Кэш дел из `ex_system` + ручной ввод №
  как фолбэк; привязка записи к `Adjudication`.
- **06 · Агент выгрузки.** Возобновляемая чанковая выгрузка, оффлайн-очередь,
  ретраи/бэкофф, статус-поллинг.
- **07 · Приём (в `ex_system`).** Django-приём записи, верификация целостности,
  передача в пайплайн W2.11 — контракт API выгрузки.
- **08 · Упаковка и импортозамещение.** Дистрибутивы по матрице
  (`.deb`/`.rpm`/NSIS·MSI/`.dmg`), объявленные системные зависимости,
  **оффлайн-установка** (бандл фикс-версии WebView2 для Windows), подпись
  пакетов (Windows code signing / Linux GPG), контролируемые обновления
  **без авто-апдейта** (ручные подписанные пакеты), CI-артефакты под матрицу и
  процедура сборки под Astra SE / РЕД ОС, чек-лист Реестра отечественного ПО.
  macOS-бандл: `Info.plist` (`NSMicrophoneUsageDescription`) и `entitlements.plist`
  (`audio-input`) для доступа к микрофону; пошаговая сборка под Ubuntu/Windows и
  сводка разрешений микрофона по ОС — `docs/packaging.md`.

### Отложено (фаза 2)

Видео (A/V-синхрон), многоканал по ролям, закладки/роли в реальном времени,
ГОСТ ЭЦП (КриптоПро), онлайн-стриминг, ALT Linux, внутренний update-канал.
